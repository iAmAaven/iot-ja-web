GAME DEVELOPMENT

My Game Dev Journey

I started developing video games in 2019 and have released seven games on itch.io. With 6 years of experience using Unity, I've become proficient in C# and object-oriented programming. Recently, I've expanded my skills by learning Godot and GDScript. My personal goal is to one day create a game that has a significant, positive, impact on the player.

Game Design

Game design is an interesting and perhaps one of the most challenging part of my development process. When making games, I try to focus on creating engaging gameplay experiences and combining fun mechanics with a compelling story. Over the years, I've learned how to build well-balanced levels, design intuitive game systems, and iterate on feedback to refine the player experience. My design philosophy revolves around ensuring that every game element adds value to the overall experience.

WordPress / PHP / REST / GraphQL

Additionally, I collaborated with team members on development tasks and maintained technical documentation throughout the project.
